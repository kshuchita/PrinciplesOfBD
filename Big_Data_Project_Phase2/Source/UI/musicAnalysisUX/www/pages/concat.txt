Code for creating table t1 from input json file containing all tweets:
./bin/spark-shell
import sqlContext.implicits._
val sqlContext = new org.apache.spark.sql.SQLContext(sc)
val t1 = sqlContext.jsonFile("newtweets.json")	
t1.registerTempTable("t1")

Code for adding 'singer' column to table t1 and saving as table t2:
import org.apache.spark.sql.functions._
val coder: (String => String) = (arg: String) => {if(arg.toLowerCase().contains("taylor")) "taylorswift" else if(arg.toLowerCase().contains("beiber")) "justinbeiber" else if(arg.toLowerCase().contains("colbertlateshow")) "justinbeiber" else if(arg.toLowerCase().contains("timberlake")) "justintimberlake" else if(arg.toLowerCase().contains("katy")) "katyperry" else if(arg.toLowerCase().contains("jlo")) "JLo"  else if(arg.toLowerCase().contains("nickiminaj")) "nickiminaj" else if(arg.toLowerCase().contains("mileycyrus")) "mileycyrus" else if(arg.toLowerCase().contains("eminem")) "eminem" else if(arg.toLowerCase().contains("bruno")) "brunomars" else if(arg.toLowerCase().contains("shakira")) "shakira" else if(arg.toLowerCase().contains("rihanna")) "rihanna" else if(arg.toLowerCase().contains("britney")) "britneyspears" else if(arg.toLowerCase().contains("gaga")) "ladygaga" else if(arg.toLowerCase().contains("selena")) "selenagomez" else if(arg.toLowerCase().contains("adele")) "adele" else ""}
val matchsinger = udf(coder)
val t2 = t1.withColumn("singer", matchsinger(col("text")))
t2.registerTempTable("t2")

Code for adding 'device' column to table t2 and saving as table tweet_table:
import org.apache.spark.sql.functions._
val coder: (String => String) = (arg: String) => {if(arg.toLowerCase().contains("iphone")) "iphone" else if(arg.toLowerCase().contains("android")) "android" else if(arg.toLowerCase().contains("ipad")) "ipad" else if(arg.toLowerCase().contains("web")) "web" else if(arg.toLowerCase().contains("windows")) "windowsphone" else if(arg.toLowerCase().contains("blackberry")) "blackberry" else ""}
val matchdevice = udf(coder)
val tweet_table = t2.withColumn("device", matchdevice(col("source")))
tweet_table.registerTempTable("tweet_table")

Code for Executing Queries:

DESCRIPTION: This query finds which are most active languages in which users tweet about the singer Taylor Swift. This will let us analyse the popularity based on language for that singer
QUERY 1: val r1 = sqlContext.sql("SELECT user.lang , COUNT(*) as cnt FROM tweet_table WHERE user.lang IS NOT NULL AND text LIKE '%taylorswift%' GROUP BY user.lang ORDER BY cnt DESC") 

DESCRIPTION: This  query finds the most used devices for tweeting by the twitter users. This will help us predict the device preferences of twitter users
QUERY 2: val r5 = sqlContext.sql("SELECT device , count(*) as cnt FROM tweet_table WHERE user.verified OR NOT user.verified GROUP BY device HAVING device != '' ORDER BY cnt DESC")
r5.coalesce(1).write.json("devices") 

DESCRIPTION: This query finds the language distribution of tweets related to music. This will help visualize which lingual section of twitter users are more interested in music.
QUERY 3: val r8 = sqlContext.sql("SELECT user.lang , COUNT(*) as cnt FROM tweet_table WHERE user.lang IS NOT NULL GROUP BY user.lang ORDER BY cnt DESC LIMIT 15")
r8.coalesce(1).write.json("all_lang") 

DESCRIPTION: This query finds the top 15 time zones from where users tweeted about singer Nicki Minaj. This will help us analyse the popularity of the singer Nicki Minaj in different time zones.
QUERY 4: val r4 = sqlContext.sql("SELECT singer, user.time_zone, SUBSTR(created_at, 0, 9),COUNT(*) AS cnt FROM tweet_table WHERE user.time_zone IS NOT NULL AND singer = 'nickiminaj' GROUP BY singer, user.time_zone,SUBSTR(created_at, 0, 9) ORDER BY cnt DESC LIMIT 10")
r4.coalesce(1).write.json("nm_location")

DESCRIPTION: This query finds the popularity of Singers based on number of tweets for each singer. This will help us visualize which singer is more popular relatively when compared to all others.
QUERY 5: val r3 = sqlContext.sql("SELECT singer ,count(*) as cnt FROM tweet_table WHERE user.verified OR NOT user.verified GROUP BY singer HAVING singer != '' ORDER BY cnt DESC")
r3.write.json("popular1")

DESCRIPTION: This query finds which Singers have tweets which could be possibly sensitive in nature. This will help us analyse what nature of tweets are written about different singers.
QUERY 6: val r10 = sqlContext.sql("SELECT singer , possibly_sensitive,count(*) as cnt FROM tweet_table WHERE possibly_sensitive GROUP BY singer, possibly_sensitive HAVING singer != '' ORDER BY cnt DESC")
r10.coalesce(1).write.json("sensitive")

DESCRIPTION: This query finds the top 15 most active regions based on number of tweets from these regions. This will help us analyse what geographic regions are active on twitter.
QUERY 7:val r2 = sqlContext.sql("SELECT user.time_zone, SUBSTR(created_at, 0, 9), COUNT(*) AS total_count FROM tweet_table WHERE user.time_zone IS NOT NULL GROUP BY user.time_zone, SUBSTR(created_at, 0, 9) ORDER BY total_count DESC LIMIT 15") 
r2.coalesce(1).write.json("timezone")

DESCRIPTION: This query finds which tweets were retweeted the most showing most popular tweets on twitter from the total retweeted count of each tweet. This will help us analyse which tweets are most popular from the time when it was first tweeted.
QUERY 8:val r6 = sqlContext.sql("SELECT COUNT(*), text as cnt FROM tweet_table GROUP BY text ORDER BY cnt DESC LIMIT 6")
r6.coalesce(1).write.json("toptweets")

DESCRIPTION: This query finds the top 15 time zones from where users tweeted about singer Taylor Swift (Second most popular singer). This will help us analyse the popularity of the singer Taylor Swift in different time zones.
QUERY 9: val r4 = sqlContext.sql("SELECT singer, user.time_zone, SUBSTR(created_at, 0, 9),COUNT(*) AS cnt FROM tweet_table WHERE user.time_zone IS NOT NULL AND singer = 'taylorswift' GROUP BY singer, user.time_zone,SUBSTR(created_at, 0, 9) ORDER BY cnt DESC LIMIT 10") 
r4.coalesce(1).write.json("ts_location")

DESCRIPTION: This query finds which tweets were most repeatedly tweeted during a period of time. This will help us analyse short term popularity of a tweet on twitter.
QUERY 10: val r6 = sqlContext.sql("SELECT COUNT(*), text as cnt FROM tweet_table GROUP BY text ORDER BY cnt DESC LIMIT 6")
r6.coalesce(1).write.json("toptweets")
